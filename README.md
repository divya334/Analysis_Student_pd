# Student Performance Analysis using Pandas

## Project Description
This project analyzes student performance data using Python, Pandas, and Matplotlib.

## Dataset Columns
- Student_ID
- Name
- Gender
- Department
- Semester
- Attendance (%)
- Internal Marks
- External Marks
- Assignment Marks
- Final Grade

## Tasks Performed
- Load dataset using Pandas
- Display dataset information and summary statistics
- Handle missing values
- Remove duplicate records
- Create Total Marks column
- Create Pass/Fail Result column
- Find Top 10 students
- Calculate department-wise average marks
- Find students with attendance below 75%
- Count students in each grade
- Visualize data using:
  - Bar Chart
  - Pie Chart
  - Histogram
  - Scatter Plot
  - Line Chart

## Technologies Used
- Python
- Pandas
- Matplotlib
- VS Code
- Git & GitHub
